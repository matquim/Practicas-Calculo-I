Prácticas Cálculo I#

El presente libro digital tiene como objetivo ofrecer actividades computacionales que puedan ser usadas en un primer curso de Cálculo. En particular, la organización se apega al temario de la asignatura de Cálculo I (Clave 1111) de la Facultad de Química (UNAM), tal que las actividades aquí mostradas sirvan como referencia para ser usadas en las clases.

La estructura de este libro busca favorecer la interactividad y el aprendizaje de conceptos. La presentación de conceptos y de código de referencia se destaca en bloques como el siguiente:

Aprendizaje de código

print("Bienvenidos a Cálculo I")

A lo largo de los ejercicios encontrará celdas vacías con una nota para indicar que debe ser llenada siguiendo alguna instrucción, como la siguiente:

# Escriba su código aquí.

En muchos casos, la celda se acompaña de una respuesta sugerida, la cual está oculta como referencia. El contenido puede revelarse como en el siguiente caso:

Show code cell content

Hide code cell content

 
print("Bienvenidos a Cálculo I, esta es la respuesta oculta a una pregunta")

Ejecución#

El código propuesto está desarrollado en Python. Con fines didácticos, se sugiere ejecutarlo en Google Colab. Como alternativa, también puede ejecutarse directamente desde la misma página web utilizando el botón Live Code, al cual se accede mediante el ícono con forma de nave espacial situado en la esquina superior derecha de las páginas.

Recuerde que puede ejecutar cada celda usando las teclas Shift + Enter, el botón run en Live Code o el triángulo de Play en Google Colab.

Le deseamos mucho éxito y una pizca de suerte a lo largo de esta travecía.
