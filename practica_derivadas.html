Derivadas#

Práctica 1: Aproximación numérica de la derivada#

Objetivo

Aproximar la derivada de una función mediante el cociente incremental.

Concepto

La derivada de una función \(f(x)\) en un punto \(x\) se define como:

()#\[\begin{equation} f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h} \end{equation}\]
def f(x):
    return x**2

x = 2
h = 0.001

derivada = (f(x + h) - f(x)) / h
print(derivada)

4.000999999999699

Ejercicios

a) Aproximar la derivada de \(f(x) = x^2\) en \(x = 2\) con distintos valores de \(h\).

b) Analizar cómo cambia la aproximación cuando \(h \to 0\).

Práctica 2: Derivadas de funciones#

Objetivo

Comparar derivadas analíticas con aproximaciones numéricas.

Ejemplo

import math

def f(x):
    return math.sin(x)

def df(x):
    return math.cos(x)

x = math.pi / 4
h = 1e-5

aprox = (f(x + h) - f(x)) / h
exacta = df(x)

print("Aprox:", aprox)
print("Exacta:", exacta)

Aprox: 0.7071032456451575
Exacta: 0.7071067811865476

Ejercicios

a) Comparar derivadas para funciones polinomiales.

b) Comparar derivadas para funciones trigonométricas.

Práctica 3: Interpretación geométrica#

Concepto teórico

La derivada representa la pendiente de la recta tangente a la curva en un punto.

Ejemplo

import numpy as np
import matplotlib.pyplot as plt

def f(x):
    return x**2

x0 = 1
m = 2*x0  # derivada exacta

x = np.linspace(-1, 3, 100)
y = f(x)

recta = m*(x - x0) + f(x0)

plt.plot(x, y)
plt.plot(x, recta)
plt.scatter([x0], [f(x0)])
plt.show()

Ejercicios

a) Graficar la función y su recta tangente en distintos puntos.

b) Interpretar el significado de la pendiente.
