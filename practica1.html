Lenguaje de programación Python#

Python es un lenguaje de programación de alto nivel (más parecido al lenguaje humano que al de la máquina). Fue creado por Guido van Rossum y la primera versión se lanzó en 1991. Es un lenguaje de propósito general diseñado para ser fácil de leer y escribir. Su sintaxis clara y sencilla lo hace ideal tanto para principiantes como para expertos.

Sus características principales son las siguientes:

  • Interpretado: No requiere compilación para ejecutarse.

  • Multiplataforma: funciona en Windows, Linux, macOS, etc.

  • Dinámico: no se necesita declarar el tipo de una variable.

  • Comunidad muy amplia y diversa: es un lenguaje de propósito general (cálculos numéricos eingeniería, desarrollo web, ciencia de datos, inteligencia artificial).

Esto se logra através del uso de diversas bibiotecas.

Bibliotecas#

  1. Biblioteca Estándar (Python Stándar Library) Es un conjunto de módulos listos para usarse sin necesidad de instalar nada. Se pueden realizar:

  • operaciones matemáticas básicas

  • Leer y escribir archivos

  • Medir tiempo y fechas

  • Generación de módulos aleatorios

  • Estadística básica

  • Expresiones regulares

  1. NumPy

Manejo eficiente de arreglos multidimensionales(similares a las matrices) y operaciones matemáticas/matriciales. Es una biblioteca base para otras como SciPy, Pandas o Scikit-learn

  1. SciPy

Funciones avanzadas para cálculo científico, optimización, integración y más. Amplía las capacidades de NumPy.

  1. Matplotlib

Para crear gráficas y visualizaciones de datos. Funciona con NumPy y Pandas

  1. SymPy

Realiza matemáticas simbólicas, como derivadas, integrales, álgebra, matrices.

  1. Pandas

Análisis y manipulación de datos en tablas (DataFrames).

  1. Scikit-learn Usada en aprendizaje automático (machine learning) para clasificación, regresión, etc.

  2. TensorFlow Inteligencia artificial y redes neuronales profundas (deep learning).

  3. Seaborn Extiende Matplotlib para hacer gráficas estadísticas más elegantes y fáciles.

  4. Cantera Simular reacciones químicas, equilibrio químico, transporet, etc.

  5. Rdkit Química computacional y quimioinformática. Soporte en modelado QSAR (Quantitative Structure-Activity Relationship)

Representación de números reales en Python#

Los números que la computadora puede representar, en particular usando el lenguaje de Python, es limitada comparada con la cardinalidad del conjunto de números reales. Por ejemplo, la representación exacta del número \(\sqrt{2}\) requiere que almacenemos una cantidad infinita de números decimales, lo mismo que los números \(\frac{1}{3}= 0.\bar{3}\), \(\frac{1}{7}= 0.\overline{142857}\), etc.

El número de bits en donde la computadora almacena información, en particular un número real, es limitado y fijo para una computadora dada. Para poder almacenar el rango más grande posible de números dado un número fijo de bits, se utilizan números de punto flotante (también llamados flotantes). En casi todas las plataformas en donde se utiliza Pyhton, los flotantes se almacenan con doble precisión (IEEE754), esto quiere decir que se designan un total de 64 bits para almacenar cada uno de ellos. Los números de punto flotante son una aproximación computacional a los números reales.

Un número flotante asigna una cantidad fija de bits a tres partes distintas de un número real:

  1. El signo, llamado \(s\), indica si es un número positivo o negativo, éste se almacena en 1 bit.

  2. Un exponente o característica, llamado \(e\), es una potencia de 2, el cual se almacena en 11 bits.

  3. La fracción o mantisa \(f\) es el coeficiente del exponente y se almacena en 52 bits.

Con esta información, un número flotante se representa de la siguiente forma:

\[\begin{equation*} n_f = (-1)^s2^{e-1023}\Big( 1+f\Big) \end{equation*}\]

En Python, la información de un número flotante se obtiene usando el paquete sys.

import sys
sys.float_info

sys.float_info(max=1.7976931348623157e+308, max_exp=1024, max_10_exp=308, min=2.2250738585072014e-308, min_exp=-1021, min_10_exp=-307, dig=15, mant_dig=53, epsilon=2.220446049250313e-16, radix=2, rounds=1)

Errores en la representación de un número real#

Algunos números racionales tampoco pueden ser representados de forma exacta en el estándar IEEE754 de números flotantes. Un ejemplo clásico de de esto es la representación de los números 0.1 y el número 0.2.

Actividad 1.3. Errores#

Muchos números recionales no tienen una representación exacta en el sistema binario que usan la computadoras para representar números en el estándar de punto flotante. Un ejemplo clásico es la representación de los números \(0.1\) y el número \(0.2\). Esto puede inducir errores en las operaciones.

Ejemplo

En un proceso se usan 0.1 L de reactivo A y 0.2 L de reactivo B. Si se producen 100 lotes al día, calcula el volúmen total usado.

volumen_total = (0.1+0.2)*1000
print(volumen_total)

300.00000000000006

#Otro ejemplo

Este tipo de errores se corrijen usando el formato Decimal en vez de float

from decimal import *

a = Decimal('0.1')
b = Decimal('0.2')
print(a+b)
print(0.1+0.2)

0.3
0.30000000000000004

Propiedades de orden de los números reales#

Para cuales quiera dos números reales, \(a\) y \(b\) sólo se cumple cualquiera de las tres afirmaciones siguientes:

()#\[\begin{align} a<b\,,\,\,\, a>b\,\,\,\, ó\,\,\,\, a=b\,,\,\,\, \end{align}\]

Python puede determinar el orden de dos números reales, es decir, determinar el valor de verdad de afirmaciones que involucren propiedades de orden.

Para esto usamos expresiones lógicas (expresiones booleanas).

Operadores de comparación.#

Tenemos los siguientes operadores de comparación:

  • ==: Igual a

  • !=: Distinto de

  • >: Mayor que

  • <: Menor que

  • >=: Mayor o igual

  • <= : Menor o igual

Al siguiente código se comparan dos números flotantes:

a = 1
b = 4
a<b

True

Estructura if#

Dado el valor de verdad de una afirmación podemos usar la estructura de ``ìf```para comparar dos números reales.

La estructura ifejecuta las líneas de código si la condición propuesta tiene un valor de True.

La sintaxis es las siguiente:
