Funciones en Python.#

Concepto

Una función de variable real es una relación entre dos conjuntos, en la cual a cada número real (del dominio de la función) se le asigna exactamente un número real (del codominio). Formalmente, una función de variable real se puede expresar como:

\(f: \mathbb D \subset \mathbb R \longmapsto \mathbb R\)

Donde \(\mathbb D\) es un subconjunto de los números reales y la función f asigna a cada número real \(x\) del conjunto \(\mathbb D\) a un solo valor real, denominado \(f(x)\).

Es muy importante recordar que las funciones de variale real se expresan algebraicamente por medio una fórmula, la cual expresa la regla de correspondencia.

En Python, podemos definir la función \(f(x) = x^2\) mediante la palabra def y la estructura:

def f(x):

    return x**2

note la palabra return que devuelve el valor de evaluar la función.

Una ventaja de hacer esto es que ahora podemos llamar a la función muchas veces sin tener que definirla de nuevo:

print("La función en x=1 vale:", f(1))
print("La función en x=2 vale:", f(2))
print("La función en x=3 vale:", f(3))

Copie y ejecute el código anterior para evaluar la función \(f(x) = x^2\) en

  • \(x=1\)

  • \(x=2\)

  • \(x=3\)

Evaluación de la función en varios puntos#

Resulta conveniente evaluar una función en varios puntos. Para ello nos auxiliaremos de la librería numpy, esta añade funcionalidades numéricas al entorno. Copie y pegue el siguiente código para importar la librería, (solo es necesario hacerlo la primera vez que se usa en la sesión de Python):

import numpy as np

# Escriba código

Show code cell content

Hide code cell content

 
import numpy as np

Usamos la función np.linspace para generar los puntos del dominio en los que se evaluará la función.

El siguiente código genera cuatro números en el intervalo del 1 al 10:

x = np.linspace(1, 10, 4)

el espaciado entre los valores se elige uniformemente acorde a la cantidad de puntos solicitados.

Copie y pegue la instrucción anterior, e imprima los valores de x.

Gráficas de funciones#

Nos resulta bastante útil usar la librería matplotlib para graficar. Para importarla, usaremos:

import matplotlib.pyplot as plt

copie y pegue el código en la siguiente celda

# Escriba código aquí

Show code cell content

Hide code cell content

 
import matplotlib.pyplot as plt

Existen dos instrucciones importantes para realizar gráficas. La primera es una gráfica de puntos (scatter). El siguiente código haría las gráficas de \(f(x) = x^2\):

x = np.linspace(-4,4, 9)
f = x**2
plt.scatter(x, f)

plt.xlabel("Eje X")
plt.ylabel("Eje Y")
plt.title("f(x) = x^2")

x = np.linspace(-4,4, 9)
f = x**2
plt.scatter(x, f)

plt.xlabel("Eje X")
plt.ylabel("Eje Y")
plt.title("f(x) = x^2")

Text(0.5, 1.0, 'f(x) = x^2')

La segunda manera es mediante el comando plot, por ejemplo:

x = np.linspace(-4,4, 9)
f = x**2
plt.plot(x, f)

plt.xlabel("Eje X")
plt.ylabel("Eje Y")
plt.title("f(x) = x^2")

Note que la gráfica no es suave. Esto se debe a que solo se usaron 9 puntos.
